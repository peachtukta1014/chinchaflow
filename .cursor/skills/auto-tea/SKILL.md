# Auto-check: ร้านชา (chincha-tea)

ตรวจสุขภาพ **อ่านอย่างเดียว** — ห้ามแก้โค้ดหรือเปิด PR ในขั้นตอนนี้

## ขั้นตอน

1. รัน build check (จาก repo root):
   ```bash
   npm run build --workspace=chincha-tea
   ```

## รายงานผล

- ✅ ผ่าน — สรุปสั้นๆ ว่าโครงสร้างปกติดี
- ❌ ไม่ผ่าน — บอกชัดว่า error อะไร ไฟล์ไหน บรรทัดไหน พร้อมแนะนำวิธีแก้

## หลังตรวจเสร็จ

ถ้าพบปัญหาและ Peach ต้องการแก้ → ใช้ `/ship-tea` ปิดงานหลังแก้แล้ว
