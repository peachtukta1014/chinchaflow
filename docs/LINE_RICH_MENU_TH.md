# Rich Menu กุ้ง (แถบล่างแชต LINE OA)

ตั้งใน [LINE Official Account Manager](https://manager.line.biz/) → ริชเมนู → **สร้างใหม่** → ตั้งเป็น **เมนูหลัก (Default)**

## แนะนำ: 3 ช่อง · Large (2500×843)

| ช่อง | ป้ายไทย | ป้าย EN (บนภาพเล็ก) | Action |
|------|---------|---------------------|--------|
| 1 | **สั่งกุ้ง** | Order | เปิด URL `https://liff.line.me/<LIFF_ID>` |
| 2 | **แชท** | Chat | ไม่ต้องเปิด URL — ใช้ข้อความสั้นใน Rich Menu ว่า “พิมพ์สั่งได้” หรือปล่อยว่าง |
| 3 | **ช่วยเหลือ** | Help | ส่งข้อความ `ช่วยเหลือ` ให้บอทตอบ (หรือ Postback → เหมือนพิมพ์ help) |

### ทำไม 3 ช่องแบบนี้

- **ไม่โล่ง** — มีทางสั่ง · ทางแชท · ทางช่วยเหลือ
- **ไม่รก** — ไม่ยัด 6 ปุ่มแบบ LINE Developers
- **มาตรฐานร้านอาหาร/ส่งของ** — สั่งหลัก · คุยรอง · ช่วยเหลือ

## ภาพพื้นหลัง (ง่ายๆ)

- สีเข้มเดียว (น้ำเงิน/เทา) + ไอคอนขาว 3 อัน
- หรือรูปกุ้งเบลอๆ ด้านหลัง + ป้ายขาวอ่านง่าย
- ขนาดไฟล์ตามที่ LINE Manager บอก (มัก 2500×843 px)

## ช่อง “แชท”

Rich Menu **ไม่มี** action “โฟกัสช่องพิมพ์” โดยตรง — ทำได้แค่:

- **ไม่ใส่ action** (กดแล้วเมนูหุบ — ลูกค้าพิมพ์เอง) ← แนะนำ  
- หรือส่งข้อความสั้น เช่น `พิมพ์สั่งได้เลยครับ` (ต้องตั้ง action ส่งข้อความ)

## ช่อง “ช่วยเหลือ”

- Action: **ข้อความ** → ลูกค้าส่ง `ช่วยเหลือ`  
- บอทตอบอัตโนมัติ (มีใน webhook แล้ว: `help` / `ช่วยเหลือ` / `menu`)

## ฟอร์ม LIFF (หลังกดสั่งกุ้ง)

1. เลือกกุ้ง + วันส่ง  
2. **คุณเคยสั่งกับทางเราไหม** → เคยแล้ว (เลือกชื่อร้าน) / ยังไม่เคย (กรอกข้อมูล)  
3. ยืนยันส่ง  

ลูกค้าที่ผูก LINE แล้ว → ข้ามขั้นชื่อ สั่งแล้วส่งได้เลย
