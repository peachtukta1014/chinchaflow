# Webhook Core Work Specs

พื้นที่นี้เก็บสเปกงานราย PR สำหรับ LINE functions / backend กลางของ CHINCHA FLOW

## Scope

- Spec ในโฟลเดอร์นี้ map กับ `apps/webhook-core/` หรือ repo config กลาง
- ใช้กับงานจาก `#chincha-flow` หรือเมื่องานระบุชัดว่าต้องแตะ LINE backend
- ห้ามแตะเว็บแอปชา/กุ้งจาก spec นี้ เว้นแต่ spec ระบุผลกระทบข้ามแอปไว้ชัดเจน
